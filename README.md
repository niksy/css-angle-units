# css-angle-units

[![Build Status][ci-img]][ci]

List of CSS [`<angle>`][angle] units.

## Install

```sh
npm install css-angle-units --save
```

## Usage

```js
var cssAngleUnits = require('css-angle-units');
console.log(cssAngleUnits);
/* [
	"deg",
	"grad",
	"rad",
	"turn"
] */
```
